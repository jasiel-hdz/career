# OFNOS

## Company Overview

OFNOS is a software company focused on building cloud-native SaaS solutions for wholesale distributors and logistics businesses.

Its products digitalize the complete distribution lifecycle, including inventory management, warehouse operations, order processing, customer management, delivery logistics, and mobile field operations.

The company emphasizes scalable cloud architectures, Infrastructure as Code, containerized deployments, and reusable software components to deliver customizable solutions for multiple customers.

## My Role

**Full Stack Software Engineer**

During my time at OFNOS, I participated in the design and development of a cloud-native SaaS platform, contributing across backend architecture, frontend development, mobile applications, cloud infrastructure, DevOps, and deployment automation.

---

## Main Project

### OFNOS SaaS Distribution Platform

A multi-tenant SaaS platform that manages the complete distribution process for wholesale companies, including inventory, warehouse operations, customer orders, delivery logistics, route optimization, and mobile applications.

The platform automatically provisions isolated cloud environments for new customers using Infrastructure as Code, allowing the same application to be deployed repeatedly with minimal manual configuration.
